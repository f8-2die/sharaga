1.Убедитесь, что у вас установлен Node.js и npm (Node Package Manager) на вашем компьютере.
Если их нет, вы можете скачать и установить Node.js с официального сайта: https://nodejs.org/

2.Откройте командную строку (терминал) и перейдите в корневую директорию вашего проекта, где находится
файл package.json.

3.Убедитесь, что все зависимости вашего проекта установлены. Выполните команду npm install
для установки зависимостей из файла package.json.

4.После установки зависимостей, вы можете собрать проект для передачи на другой компьютер. 
Для этого используйте команду npm run build. Эта команда создаст сборку проекта в папке dist 
(или другой папке, указанной в конфигурации проекта).

5.Архивируйте папку dist, которая содержит собранный проект. Вы можете использовать архиватор,
 например, ZIP, чтобы создать архив с содержимым папки dist.

6.Передайте архив (ZIP-файл) другому компьютеру, на который вы хотите установить и запустить проект.

7.На другом компьютере распакуйте архив с проектом.

8.После распаковки, перейдите в директорию проекта и выполните команду npm install для установки
зависимостей, если это не было сделано ранее.

9.Затем вы можете запустить проект с помощью команды npm start или другой команды, предусмотренной
в вашем файле package.json для запуска проекта.


node server.js
npm run serve
